# Extension Boilerplate

A minimal browser extension starter.

## Included

- Manifest V3 starter
- Firefox manifest starter
- Popup, background, and content-script placeholders
- Cross-browser `chrome` / `browser` shim

## Next steps

1. Rename the extension in `manifest.json` and `manifest.firefox.json`
2. Replace the popup UI in `src/popup.html` and `src/popup.js`
3. Add your feature logic to `src/main.js` and `src/background.js`
4. Remove unused files once you know your target browser and feature set
